## 🧠 Project Overview

This is a high-performance, futuristic personal portfolio website designed and developed by **Ritesh Kumar**, showcasing expertise in Artificial Intelligence, Machine Learning, and Intelligent Systems Architecture.

The portfolio goes beyond traditional static designs by integrating:

- ⚡ Interactive system-style UI  
- 🧬 Neural network-inspired animations  
- 🖥️ Terminal-based storytelling experience  
- 🌌 Glassmorphism + cyberpunk aesthetic  

It presents projects, skills, achievements, and experience in a visually immersive and technically advanced format.

> Built to simulate a digital operating system interface (RITESH.OS).

---

## ✨ Features

### 🎨 UI / UX Highlights
- 🧊 Advanced Glassmorphism Design with dynamic lighting  
- 🌗 Dark/Light Mode Toggle with localStorage  
- 🎯 Custom Animated Cursor (LERP-based)  
- 💡 Spotlight Hover Effects  
- 🔥 Neon cyberpunk theme  

### ⚙️ Interactivity & Animations
- 🎬 GSAP + ScrollTrigger animations  
- 📜 AOS (Animate On Scroll)  
- ⌨️ Typing animation (Typed.js)  
- 🧠 Neural Network Canvas Background  
- 🖥️ Live Terminal Simulation  
- 🕹️ Easter Egg Terminal (press `~`)  

### 📱 Responsiveness
- Fully responsive (Mobile, Tablet, Desktop)  
- Mobile navigation menu  

### 🧩 Sections Included
- About  
- Skills  
- Projects  
- Achievements  
- Experience  
- Education  
- Contact
